# About The Lennons Wiki

Welcome to my personal wiki and technical documentation hub!

## What is This Site?

This is a collection of guides, tutorials, and technical documentation that I've created while working on various projects. The goal is to share knowledge and help others who might encounter similar challenges.

## Topics Covered

### Homelab & Infrastructure
- Proxmox virtualization setup and configuration
- Network storage and NAS security
- Self-hosted services and applications

### IoT & Smart Home
- Zigbee device integration and troubleshooting
- DIY electronics projects
- Home automation solutions

### IT & Troubleshooting
- Windows system fixes and configurations
- General technical support guides
- Software and hardware troubleshooting

## Why I Created This

I started this wiki to:
- Document solutions to problems I've encountered
- Create a reference for my future self
- Share knowledge with the community
- Practice technical writing and documentation
